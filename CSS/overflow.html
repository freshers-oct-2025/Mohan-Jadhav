visible

This is an example of overflow: visible. The extra text will overflow outside the box and still be visible to you.

hidden

This is an example of overflow: hidden. The extra text is clipped and not visible outside the box.

scroll

This is an example of overflow: scroll. Scrollbars are always visible, even if content fits inside.

auto

This is an example of overflow: auto. Scrollbars appear only when the content overflows the box.
